\begin{frame}[t]{同時更新}{更新が競合したら}

  local\_rep と clone\_rep で同時更新
  \vspace{4ex}

  \onslide*<1>{
    local\_rep で更新
    \vspace{2ex}

    \$ cd \~{}/local\_rep

    \$ touch h

    \$ git add h

    \$ git commit -m 'Add h.'

    \$ git push origin master
  }

  \onslide*<2>{
    clone\_rep で更新
    \vspace{2ex}

    \$ cd \~{}/clone\_rep

    \$ touch i

    \$ git add i

    \$ git commit -m 'Add i.'

    \$ git push origin master (失敗)
  }

  \onslide*<3>{
    clone\_rep で local\_rep の更新を取り込み、再 push
    \vspace{2ex}

    \$ git fetch {\dhyphen}all -p

    \$ git merge remotes/origin/master

    \$ git push origin master
  }

  \onslide*<4>{
    local\_rep に clone\_rep の更新を取り込む
    \vspace{2ex}

    \$ cd \~{}/local\_rep

    \$ git fetch {\dhyphen}all -p

    \$ git merge remotes/origin/master
  }

\end{frame}
